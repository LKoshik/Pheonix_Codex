## Adding a New Philosophy

1. Write your new philosophy as a standalone HTML file (e.g. `scroll-iii.html`)
2. Add the nav bar at the top (copy from any existing scroll)
3. Open `index.html` and add a new card in the `.cards-grid` section
4. Commit and push — it's live
